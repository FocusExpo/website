# FOCUS Expo Landing Page

A premium landing page for the FOCUS Expo event, built with React, Vite, and Framer Motion.

## Project Structure

- `src/components/`: Individual section components (Hero, Intro, Pillars, etc.)
- `src/styles/`: Global styles (though currently using `index.css` and `Hero.css`)
- `src/assets/`: Images and SVGs from the design package

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Deployment

This project is configured for Netlify deployment via `netlify.toml`.
Simply connect your repository to Netlify, and it should auto-detect the settings.

## Design

- **Typography**: Outfit (Headings), Inter (Body)
- **Colors**: Slate 900 (Background), Blue 500 (Primary), Amber 500 (Secondary)
- **Animations**: Framer Motion for scroll reveals
- **Assets**: Custom geometric patterns and icons
